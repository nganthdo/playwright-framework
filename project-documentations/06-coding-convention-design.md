## Coding Convention

### Naming
#### File Naming:
    - Test file: *.spec.ts
    - Page: *.page.ts
    - Fixture: *.fixture.ts
    - API: *.api.ts
- Keep naming consistent(test, page, api)
    - Example:
        - post.spec.ts
        - post.page.ts
        - post.api.ts

#### Code Naming:
    - camelCase cho function (e.g., loginUser, createPost)
    - PascalCase cho class (e.g., LoginPage, DashboardPage)
